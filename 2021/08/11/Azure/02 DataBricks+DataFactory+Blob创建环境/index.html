02 DataBricks+DataFactory+Blob创建环境

发表于 2021/08/11 | 分类于 Azure
字数统计: 270

本文主要是介绍创建Blob\Databricks\Datafactory资源

Azure

进入Azure 首页https://portal.azure.cn/?whr=azure.com#home 点击创建资源。如下图

填入相应信息，直接点击审阅并创建即可。创建好点击进入在blob中创建一个blob 如data_file

DataBrikcs

DataBricks创建过程与Blob基本相同。注意创建DataBricks后要创建一个集群用来计算处理数据，执行DataBricksjob。

集群配置选择正常的开发配置就可以。下图是当前项目集群配置

注意databriks是计算工具不是存储工具。

创建Databricks访问令牌,进入databricks 点击用户名进入User Setting.生成token,设置token名称和生效时长。注意保存token

DataFactory

DataFactory创建过程与Blob基本相同。在创建好DataFactory需要配置一些链接，如访问Blob的链接、访问DataBriks的链接、访问s3容器的链接、访问Delta Lake的链接。

创建Databricks链接是需要填写一些配置，主要是访问令牌

创建Delta Lake的链接与创建DataBricks链接的过程类似。
